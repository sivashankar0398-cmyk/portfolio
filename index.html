The Clinical Protocol State Machine

Multi-agent engine validating dense, unstructured trial configurations against rulesets.

GitHub Source

The Problem & Architecture

In clinical trials, data managers manually map guidelines across 300-page PDF clinical protocols. Our multi-agent architecture utilizes LangGraph state routers. The pipeline employs Agent 1 to extract rules and map dependencies. This data routes to Agent 2 to synthesize test scenarios. Agent 3 then processes validations in a secure sandbox.

Failure Modes & Mitigation

A primary failure mode involved token overflow and loop cycling during dense structural analyses. We resolved this by defining structured state-boundary limits and implementing thread-safe execution gates inside our LangGraph routers, ensuring clean state persistence.

from typing import TypedDict, List, Dict, Any
from langgraph.graph import StateGraph, END

# Define robust, serializable state schema
class ClinicalReviewState(TypedDict):
    protocol_id: str
    raw_document_url: str
    extracted_rules: List[Dict[str, Any]]
    synthetic_test_scenarios: List[Dict[str, Any]]
    sandbox_execution_logs: List[str]
    loop_count: int
    validation_status: bool

def agent_extract_rules(state: ClinicalReviewState) -> Dict[str, Any]:
    # Extract structural dependencies via Claude-3.5 Sonnet Tool Calling
    rules = [{"rule_id": "R101", "type": "exclusion_criteria", "score": 0.98}]
    return {"extracted_rules": rules, "loop_count": state["loop_count"] + 1}

def state_gate_router(state: ClinicalReviewState) -> str:
    # Strict loop mitigation boundary checker
    if state["loop_count"] > 3:
        return "human_in_the_loop"
    if len(state["extracted_rules"]) > 0:
        return "synthesize_scenarios"
    return END

# Initialize State Machine Topology
workflow = StateGraph(ClinicalReviewState)
workflow.add_node("extractor", agent_extract_rules)
workflow.set_entry_point("extractor")
workflow.add_conditional_edges("extractor", state_gate_router, {
    "human_in_the_loop": END,
    "synthesize_scenarios": END
})
app = workflow.compile()
